# fwnet

<https://fw.ac.cn/>

![fwnet logo](https://fw.ac.cn/fw-banner.svg)

</div>

---

fwnet 是一个去中心化的实验性网络社区，由爱好者共同构建和维护。  

简而言之，fwnet 通过先进的网络技术（包括基于 BGP 协议的交换中心和 RPKI 路由验证）创建了一个安全、高效且具有弹性的私有网络环境，为成员提供类似互联网基础设施的完整体验，同时保持高度的自治性和技术探索空间。

---

CN86 已通过以下途径与 fwnet 进行互联，成为 [其成员](https://fw.ac.cn/other-projects#cn86) 的同时保有高度自治权：

| CN86 设施 | fwnet 设施 | 协议 |
| --------- | ---------- | ---- |
| 宁波节点  | vIX2       | BGP  |

## ASN 

fwnet 使用私有 ASN 4237370000 ~ 4237379999。

## DNS

fwnet 主要使用私有后缀 `.fwnet`，并设有两个 NS 服务器。

## IP

fwnet 分配了繁多的 IP 段，若要了解请移步至 [fw.ac.cn](https://fw.ac.cn/)。
