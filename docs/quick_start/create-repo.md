# 创建你的资料仓库
资料仓库是你在CN86网络中的身份卡  

> [!WARNING]
> 请将仓库设为公开，请不要fork他人的仓库

## 步骤
 1. 点击右上角的 **"+"** 按钮
 2. 选择创建仓库
 3. 在新打开的仓库中配置信息
    - 拥有者：`你自己的用户名`
    - 仓库名称：`mntner`（请确保完全一致）
    - 其它配置项请保持默认
 4. 点击 **创建仓库**
 5. 至此仓库创建完成
